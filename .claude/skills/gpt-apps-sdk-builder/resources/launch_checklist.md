# 公開前チェックリスト

## 接続・動作

- /mcp が 200 または適切なレスポンスを返す
- OPTIONS のCORSプリフライトが成功する
- tools/call が期待する structuredContent を返す
- UIがある場合は resourceUri が一致している

## セキュリティ

- 機密情報は環境変数で管理されている
- 認証が必要な場合は認可フローが機能する
- ログに個人情報が出ない

## 体験

- ツール名と説明が短く明確
- エラーメッセージが理解しやすい
- UI操作が少ない手順で完結する
